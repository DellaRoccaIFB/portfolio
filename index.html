IFB ACCESS

IFB ACCESS é um projeto desenvolvido que oferece uma solução completa para modernizar e reativar o sistema de controle de acesso do Instituto Federal de Brasília (IFB), que se encontrava inoperante. Foi implementado solução baseada em NFC (Near Field Communication) para permitir que a comunidade acadêmica utilize seus próprios dispositivos móveis para acessar a instituição.

Android, Near Field Communication(NFC), Kotlin, Jetpack Compose, Firebase, Docker, Docker Compose, Ngrok, Nginx, Rest API, Flask, Spring Boot, MySQL, Arduino
